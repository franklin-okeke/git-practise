BRANCHING

git branch dev  - creates a branch called dev
git branch feature/contact-form  - creates a branch called feature/contact-form

NB: always pull the latest chnages before creating a branch

git checkout dev - navigates to the branch dev
git checkout -b dev  - creates and navigates inside the branch once

git branch -- shows us all the branch we r have in the project

git pull origin main - pulls the lattest chnages from the main branch
git pull origin dev - pulls the lattest chnages from the dev branch
